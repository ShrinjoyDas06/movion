## 🚀 About Movion

**Movion** is a modern mobile app for movie lovers.  
Browse trending films, explore detailed information, and build your personal watchlist — all in one smooth and elegant experience.  

Whether you’re discovering your next favorite movie or keeping track of what you’ve already seen, **Movion** makes your cinematic journey effortless.

📱 **Download APK:** [Click here](#)

---

## 🛠️ Tech Stack

**Frontend:** React Native + Expo  
**Routing:** Expo Router  
**State Management & Storage:**
- 🧠 **React Context / Hooks** for global state
- 💾 **AsyncStorage** for persistent local data (Saved & Watched movies)

**UI & Styling:**
- 🎨 **Expo Linear Gradient** for stylish backgrounds  
- 🔔 **Toastify React Native** for notifications  
- 🧩 **Lucide React Native** for icons  

**Animation & Performance:**
- ⚡ **React Native Reanimated**  
- ✋ **React Native Gesture Handler**  

**Typography:**
- 🅰️ **Expo Font** for custom fonts and modern aesthetics

## ✨ Features

### 🎞️ Movie Gallery
A stunning grid of movie posters that you can tap to view detailed information.

### 📖 Movie Details
Get everything you need before hitting play:
- 🎬 Title & Plot Summary  
- 🏆 Awards & IMDb Rating  
- 🎭 Genre, Year, Cast, Screenplay  
- 🕒 Runtime, Language, and more  

### ❤️ Save for Later
Save movies you want to watch later. Stored locally using AsyncStorage.

### ✅ Mark as Watched
Keep track of the movies you’ve already watched.

### 🔔 Notifications & Feedback
Toast messages give real-time confirmation for actions like saving, removing, or marking movies as watched.

### 📤 Share Your Favorites
Share any movie with friends via clipboard or social media.

### 💫 Smooth Animations
Fluid transitions, gesture interactions, and visual polish throughout the app.

## ⚙️ Installation Guide

### 📦 Install the APK
1. **Download the APK:** [Click here](#)
2. **Enable Install from Unknown Sources:**
   - Open **Settings → Apps & Notifications → Special Access → Install Unknown Apps**
   - Choose your browser or file manager and enable **Allow from this source**
3. **Install the App:**
   - Locate the APK in your file manager  
   - Tap and follow installation steps  
4. Find **Movion** in your app drawer and start exploring!
